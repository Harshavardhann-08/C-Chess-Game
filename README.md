# C++ Chess Game

A two-player console-based chess game built in C++ using object-oriented programming.

## Overview

This project implements a local two-player chess game with board creation, piece movement, move validation, turn handling, check detection, checkmate detection, and stalemate detection.

The project was developed for ENGTECH 1PR3 - Object Oriented Programming at McMaster University.

## Features

- 8x8 chess board representation
- Two-player local gameplay
- Turn-based movement
- Legal move validation
- Check detection
- Checkmate detection
- Stalemate detection
- Object-oriented piece structure using inheritance

## Technologies Used

- C++
- Object-Oriented Programming
- Classes and Inheritance
- Pointers
- Dynamic memory management
- Console input/output
